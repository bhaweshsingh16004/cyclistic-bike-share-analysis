# cyclistic-bike-share-analysis
Google Data Analytics Capstone Project

## Project Overview
This project analyzes differences between casual riders and annual members using Cyclistic bike data.

## Tools Used
Google Sheets
Pivot Tables
Data Cleaning
Charts

## Data Source
Divvy Bike Share public dataset.

## Business Question
How do casual riders and annual members use bikes differently?

## Key Findings
Members ride mainly during weekdays.
Casual riders prefer weekends.
Casual riders have longer average ride times.

## Recommendations
- Increase marketing during peak usage periods
- Target casual users for membership conversion
- Optimize pricing strategy based on usage trends
